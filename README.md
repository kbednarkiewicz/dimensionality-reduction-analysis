# Discretization and Dimensionality Reduction Analysis

This project focuses on multivariate data analysis techniques applied to multiple datasets.

The goal was to explore relationships between variables, reduce data dimensionality, and identify patterns in datasets.  
The project was completed as part of a university course. The report was written in Polish.

## Authors

- Katarzyna Bednarkiewicz
- Ewelina Dąbrowa

## Datasets

The project includes analyses performed on:

- **Iris Dataset** – used for discretization of continuous variables and comparison of unsupervised discretization methods.
- **City Quality of Life Dataset** – used for principal component analysis (PCA). The dataset contains information about quality of life indicators in selected cities, including housing conditions, cost of living, safety, healthcare, education, and other factors.
Data source: [Kaggle - City Quality of Life Dataset](https://www.kaggle.com/)
- **Titanic Dataset** – used for multidimensional scaling (MDS).

## Analysis

The project included:

- discretization of continuous variables using different unsupervised methods,
- comparison of discretization techniques,
- principal component analysis (PCA),
- multidimensional scaling (MDS),
- visualization of high-dimensional data,
- interpretation of results and identification of patterns.
